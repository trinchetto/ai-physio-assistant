# DALL-E 3 System Prompt for Exercise Illustrations

Use this system prompt when creating a Custom GPT or at the start of a ChatGPT session.

---

## System Prompt (Copy This)

```
You are an assistant that generates exercise illustrations for a physiotherapy application.

STYLE REQUIREMENTS (apply to ALL images):
- Simple line drawing illustration style
- Black lines on pure white background
- Gender-neutral human figure with anatomically accurate proportions
- Physical therapy instruction diagram aesthetic (like medical textbook illustrations)
- Minimal detail, clean lines, no shading
- No colors except black lines on white
- NEVER include text, labels, numbers, letters, or watermarks
- Full figure visible, not cropped (unless specifically a close-up)
- Single figure per image (unless comparison is requested)

WHEN I DESCRIBE AN EXERCISE POSITION:
1. Generate an image following the style requirements above
2. Show the exact position/movement I describe
3. Use the viewing angle I specify (side view, front view, etc.)
4. Keep the style consistent with previous images in this session

If I say "side view" - show the figure from the side (profile)
If I say "front view" - show the figure facing the viewer
If I say "angle view" - show from a 3/4 angle
If I say "close-up" - show only the relevant body part

Ready to generate exercise illustrations.
```

---

## How to Use

### Option 1: Custom GPT (Recommended)
1. Go to ChatGPT → Explore GPTs → Create
2. Name it "Exercise Illustrator" or similar
3. Paste the system prompt in the Instructions field
4. Save and use for all image generation

### Option 2: Start of Session
1. Paste the system prompt at the beginning of a new ChatGPT conversation
2. ChatGPT will acknowledge the instructions
3. Then use the simplified prompts below

## Simplified Prompts

With the system prompt active, you can use these shorter prompts:

### CHIN TUCK

**1. Start**
```
Side view: Person sitting upright on a chair, good posture, head in neutral position, looking straight ahead, shoulders relaxed.
```

**2. Retracted**
```
Side view: Person sitting on chair, chin drawn straight back creating double chin, head level (not tilting), ears moving back toward shoulders.
```

**3. Comparison**
```
Two side-view heads: Left shows forward head posture with chin jutting forward. Right shows corrected position with chin tucked back, ears aligned over shoulders.
```

---

### PENDULUM EXERCISE

**1. Start**
```
Side view: Person bent forward at waist (90 degrees), one hand on table for support, other arm hanging straight down completely relaxed toward floor.
```

**2. Circular**
```
Front view: Person bent forward, hand on table, hanging arm shown with dotted circle indicating small circular swinging motion.
```

**3. Swing**
```
Side view: Person bent forward, hand on table, hanging arm shown with dotted lines indicating forward-backward pendulum swing motion.
```

---

### CAT-COW STRETCH

**1. Tabletop**
```
Side view: Person on hands and knees, back flat and neutral, wrists under shoulders, knees under hips, head looking slightly down.
```

**2. Cat**
```
Side view: Person on hands and knees in cat pose - spine rounded up toward ceiling like an arching cat, head dropped down, tailbone tucked under.
```

**3. Cow**
```
Side view: Person on hands and knees in cow pose - belly dropped toward floor, back gently arched, head lifted looking forward, tailbone tilted up.
```

---

### PIRIFORMIS STRETCH

**1. Start**
```
Angle view from above: Person lying on back, both knees bent, feet flat on floor, arms relaxed at sides.
```

**2. Figure-4**
```
Angle view from above: Person lying on back, one ankle crossed over opposite knee creating figure-4 shape, foot flexed.
```

**3. Full Stretch**
```
Side view: Person lying on back doing figure-4 stretch, hands clasped behind thigh of bottom leg pulling toward chest, back stays flat on floor.
```

---

### CALF RAISES

**1. Start**
```
Side view: Person standing upright, feet hip-width apart, one hand lightly touching wall for balance, heels flat on ground, body straight.
```

**2. Raised**
```
Side view: Person standing on tiptoes, heels raised high off ground, hand on wall for balance, body straight and vertical.
```

**3. Foot Detail**
```
Close-up side view: Foot and lower leg only, performing calf raise - heel raised high, weight on ball of foot, calf muscle slightly visible.
```

## Troubleshooting

| Problem | Add to prompt |
|---------|---------------|
| Image has text/labels | "Remember: no text or labels" |
| Style inconsistent | "Match the style of previous images" |
| Figure cut off | "Show full figure, not cropped" |
| Too much detail | "Simpler, fewer lines" |
| Wrong angle | Explicitly state: "side view" / "front view" |
| Background not white | "Pure white background" |

---

## Session Workflow

1. **Start session**: Paste system prompt
2. **Generate by exercise**: Do all 3 images for one exercise before moving to next
3. **Review each image**: Regenerate if needed before moving on
4. **Download immediately**: Save with correct filename
5. **Note variations**: If a prompt needed adjustment, update for future use
