## Tips for Best Results

1. **If image has text/labels**: Add "absolutely no text, no labels, no numbers, no letters" to prompt

2. **If style is inconsistent**: Generate all 3 images for one exercise in same chat session

3. **If figure is cut off**: Add "full figure visible, not cropped"

4. **If too detailed/realistic**: Add "minimal detail, simple clean lines"

5. **If background not white**: Add "pure white background, no shadows"
